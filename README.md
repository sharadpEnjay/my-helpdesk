# Bun Fullstack App

A modern fullstack application built with Bun, Express, React, and TypeScript.

## Project Structure

- `/server`: Express backend running on Bun.
- `/client`: React frontend built with Vite.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) installed on your system.

### Installation

From the root directory:

```bash
bun install
```

### Development

To start both the server and the client in development mode:

```bash
bun run dev
```

- Server will be at: `http://localhost:3001`
- Client will be at: `http://localhost:5173`

## Tech Stack

- **Runtime**: Bun
- **Backend**: Express + TypeScript
- **Frontend**: React + Vite + TypeScript
- **Styling**: Vanilla CSS (Glassmorphism design)
- **Monorepo Management**: Bun Workspaces
